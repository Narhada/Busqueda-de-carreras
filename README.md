# Busqueda-de-carreras

Un sitio donde puedas consultar carreras en michoacán, colocando el nombre de lo
que quieres estudiar y dando las diferentes carreras que puedan contener esa materia,
por ejemplo el nombre de nuestra carrera lo podríamos relacionar con alguna otra buscando
unas palabras o materias en común, esto servirá para encontrar una carrera que probablemente
sea algo parecido a lo que quieren estudiar. Ofrece información clave, como lo son las fechas
de admisión, características de las licenciaturas de interés entre otras cosas.
